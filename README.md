# H5P Interactive Book

Interactive Book is an H5P-component, giving users a central knowledge base which utilizes the [H5P Framework](https://github.com/h5p)

Interactive Book "papi Jo" (the H5P.InteractiveBookPapiJo library) is a modified version which accepts the "papi Jo" versions of Dialog Cards and Drag & Drop activities.
It needs the "papi Jo" version of Column to be installed first (H5P.ColumnPapiJo library).

## Getting started

Grab all the modules:

```bash
npm install
```

Build project:

```bash
npm run build
```
